# Learning Java — Payments & AI Engineering

A structured Java learning environment for a Senior Salesforce Engineer
transitioning to payment processing and AI engineering at Checkout.com.

## Quick Start

```bash
cd learning-java
# Start a learning session
# Tell Claude: "start session"
```

## Structure

- `CLAUDE.md` — Full curriculum, rules, and learning philosophy
- `progress.json` — Current learning state (auto-updated)
- `learning-log.md` — Session journal
- `challenges/` — Practice problems per module
- `projects/` — Build projects per phase

## Commands

- `start session` — Load progress and begin
- `give me a problem` — Get a challenge for current module
- `I'm stuck` — Get guided help (no spoilers)
- `I solved it` — Mark complete and advance
- `end session` — Save progress and log

## Prerequisites

- Java 21+ (LTS) — `brew install openjdk@21`
- Gradle 8+ — `brew install gradle`
- Neovim with jdtls LSP configured
